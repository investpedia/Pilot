Comments:
Multi-line comments start with the /* pair of symbols and end with the */ one. 
Such kind of comments cannot be nested. 
Single-line comments begin with the 
// pair of symbols and end with the newline character, 
// they can be nested in other multi-line comments. 
// Comments are allowed everywhere where the spaces are allowed, 
// they can have any number of spaces in them.

Examples:

//--- Single-line comment
/*  Multi-
    line         // Nested single-line comment
    comment
*/
